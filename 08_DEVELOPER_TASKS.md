# Developer Tasks

## Phase 1: Fundament
1. Shop-System auswählen
2. Theme/Frontend-Struktur festlegen
3. Seitenstruktur anlegen
4. Rechtliche Pflichtseiten vorbereiten
5. Produktkategorien definieren

## Phase 2: Conversion Frontend
1. Startseite mit Hero und Bestseller bauen
2. Produktgrid optimieren
3. Produktdetailseite mit Sticky CTA erstellen
4. Mobile Navigation bauen
5. Warenkorb und Checkout vereinfachen

## Phase 3: Produkte
1. Produktbilder optimieren
2. Varianten anlegen
3. Größeninformationen einpflegen
4. Produkttexte einfügen
5. Upsell-/Cross-Sell-Produkte zuweisen

## Phase 4: Tracking & Marketing
1. GA4 einrichten
2. Meta Pixel einrichten
3. TikTok Pixel einrichten
4. Newsletter-System integrieren
5. Rabattcode für Erstkäufer erstellen

## Phase 5: Launch-Test
1. Testkauf durchführen
2. Zahlung testen
3. Versandmails testen
4. Mobile Speed testen
5. Checkout-Abbruch testen
6. Fehler beheben

## Phase 6: Optimierung nach Launch
1. Heatmaps analysieren
2. Warenkorbabbrüche prüfen
3. Bestseller hervorheben
4. Produktbilder A/B testen
5. Hero Headline A/B testen
