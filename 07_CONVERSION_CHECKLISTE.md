# Conversion-Checkliste

## Startseite
- [ ] Hero CTA sichtbar ohne Scrollen
- [ ] Bestseller direkt auf der Startseite
- [ ] Trust-Hinweise sichtbar
- [ ] Mobile Darstellung getestet
- [ ] Ladezeit optimiert

## Produktseite
- [ ] Produktbilder hochwertig
- [ ] Preis klar sichtbar
- [ ] Größenwahl verständlich
- [ ] CTA immer sichtbar auf Mobile
- [ ] Versand- und Rückgabehinweise sichtbar
- [ ] Größentabelle vorhanden
- [ ] Bewertungen integriert

## Warenkorb
- [ ] CTA „Zur Kasse“ auffällig
- [ ] Versandhinweis sichtbar
- [ ] Keine unnötige Ablenkung
- [ ] Rabattcode-Feld nicht zu dominant

## Checkout
- [ ] Gastbestellung möglich
- [ ] Express-Zahlungen aktiv
- [ ] Möglichst wenige Pflichtfelder
- [ ] Fortschrittsanzeige vorhanden
- [ ] Trust-Elemente sichtbar

## Tracking
- [ ] ViewContent Event
- [ ] AddToCart Event
- [ ] InitiateCheckout Event
- [ ] Purchase Event
- [ ] Meta Pixel getestet
- [ ] GA4 getestet

## Launch
- [ ] Testbestellung durchgeführt
- [ ] Mobile Test auf mehreren Geräten
- [ ] Rechtliche Seiten geprüft
- [ ] Versandkosten korrekt
- [ ] Zahlungsarten funktionieren
- [ ] E-Mail-Bestätigungen funktionieren
